## 📌 О проекте

Интерактивный дашборд для анализа текучести персонала (turnover) в Power BI. Проект демонстрирует навыки:

- **Анализ KPI** — расчёт ключевых метрик текучести
- **Факторный анализ** — выявление драйверов увольнений
- **Визуализацию** — интерактивные графики и фильтры
- **Бизнес-рекомендации** — формулирование выводов и предложений по снижению turnover

## 💡 Выводы и рекомендации

### 🔴 Ключевые факторы текучести (Key Turnover Drivers)
1. **Переработки** — риск увольнения выше в 3 раза (30,6% vs 10,4%)
2. **Молодые сотрудники (18–25 лет)** — текучесть 35,8%
3. **Департамент Sales** — текучесть 20,7% (выше среднего на 29%)
4. **Должность Sales Representative** — текучесть 39,3%
5. **Женщины** — текучесть 22,1% (выше, чем у мужчин 14,5%)

### 🟢 Рекомендации по снижению текучести (Recommendations to Reduce Turnover)
1. Пересмотреть политику переработок в департаменте Sales
2. Внедрить программу адаптации для молодых сотрудников (18–25 лет)
3. Провести анализ причин увольнения женщин
4. Добавить в 1С поля:
   - «Причина увольнения» (для точного факторного анализа)
   - «Стаж работы с текущим руководителем»
   - «Удовлетворённость работой»

### 📌 Дальнейшие шаги по автоматизации
- Настроить ежемесячную рассылку дашборда руководителям департаментов
- Автоматизировать сбор данных о переработках из системы учёта рабочего времени
- Создать автоматическую выгрузку данных из 1С в Power BI

## 🛠️ Используемые инструменты

| Инструмент | Назначение |
|------------|------------|
| **Power BI Desktop** | Создание дашборда, визуализация данных |
| **Power Query** | Очистка и преобразование данных |
| **DAX** | Создание мер для расчёта KPI |
| **Excel / Python** | Генерация и подготовка данных |

## 🚀 Как использовать

1. Скачайте файл `HR_Analytics_Dashboard.pbix`
2. Откройте в Power BI Desktop
3. Используйте срезы (фильтры) для интерактивного анализа:
   - Департамент
   - Пол
   - Переработки
   - Возрастная группа
